### 1) التقنية والقيود

* **الإطار:** Next.js 15 (App Router) + TypeScript.
* **الواجهة:** Tailwind CSS + shadcn/ui + Framer Motion للموشن.
* **الأيقونات:** lucide-react.
* **البناء والتسليم:** SSG كافتراضي، عدم الاعتماد على أي قاعدة بيانات.
* **i18n:** next-intl أو next-i18next مع ملفات JSON للمفاتيح. جميع المفاتيح باسماء A1..A200+ كأرقام متسلسلة تحت حرف A حصراً. لغتان على الأقل: `ar`, `en`.
* **SEO Ultra Pro:** next-seo + sitemap + robots + OG/Meta + JSON-LD منظّم.
* **الأمان:** CSP صارم، Security headers، منع XSS، تنظيف المدخلات.
* **الأداء:** صور Next/Image، lazy, prefetch ذكي، ضغط وminify، قيود أداء: LCP < 1.8s على شبكة 4G، CLS ~0، TBT منخفض.
* **الوصولية:** A11y صارم، تباين صحيح، اختصارات لوحة مفاتيح للأقسام التفاعلية.
* **عدم وجود DB:** كل المحتوى ثابت داخل `/content` وملفات ترجمة JSON.

### 5) التوجّه المعماري

* **Localization routing:** مجلد `[locale]` مع middleware لتحديد اللغة الافتراضية من Accept-Language وسقوط إلى `en`.
* **Content sourcing:** جميع النصوص من ملفات JSON أعلاه. لا مفاتيح صريحة داخل JSX. استخدم hook مترجم.
* **التفاعل:** HeroShowcase يقرأ/يكتب حالة عرض محليّة فقط. لا كookies ولا تخزين.
* **CSS:** مكونات shadcn مصقولة. قواعد Tailwind في `themes.css`. دعم RTL تلقائي عند `ar`.

### 8) نموذج التواصل + Resend

* **سير العمل:**

  1. POST `/api/contact` يستقبل `{ name, email, phone, message, locale }`.
  2. يولّد رقم تذكرة بصيغة: `VTP-${YYYYMMDD}-${XXXX}` حيث XXXX رقم عشوائي 4 أرقام.
  3. يرسل رسالتين عبر Resend:

     * إلى العميل: “تأكيد استلام” مع رقم التذكرة وقائمة مختصرة بالبيانات.
     * إلى بريد الشركة: إشعار داخلي شامل.
  4. يستجيب JSON: `{ ok: true, ticket }`.
* **البيئة:** لا تضع المفاتيح في الكود. استخدم:

  ```
  RESEND_API_KEY=
  COMPANY_EMAIL=support@vetaps.com
  COMPANY_NAME=VETAP
  FROM_EMAIL=VETAP <no-reply@vetaps.com>
  ```
* **التحقق:** Zod للتحقق من الحقول، قيود طول، sanitize أساسي، معدل بسيط بالذاكرة لمنع الإساءة.

مثال `app/api/contact/route.ts` مختصر وفعّال:

```ts
import { NextRequest, NextResponse } from 'next/server';
import { z } from 'zod';
import { Resend } from 'resend';
import { renderClientEmailHTML, renderCompanyEmailHTML } from '@/lib/mail';
import { genTicket } from '@/lib/ticket';

const schema = z.object({
  name: z.string().min(2).max(80),
  email: z.string().email(),
  phone: z.string().min(5).max(25).optional(),
  message: z.string().min(10).max(2000),
  locale: z.enum(['ar','en']).default('en')
});

const resend = new Resend(process.env.RESEND_API_KEY!);

export async function POST(req: NextRequest) {
  try {
    const body = await req.json();
    const data = schema.parse(body);
    const ticket = genTicket();

    // HTMLs
    const clientHTML = renderClientEmailHTML({ ...data, ticket });
    const companyHTML = renderCompanyEmailHTML({ ...data, ticket });

    await Promise.all([
      resend.emails.send({
        from: process.env.FROM_EMAIL!,
        to: data.email,
        subject: `VETAP • ${data.locale === 'ar' ? 'تأكيد الطلب' : 'Request Confirmation'} • ${ticket}`,
        html: clientHTML
      }),
      resend.emails.send({
        from: process.env.FROM_EMAIL!,
        to: process.env.COMPANY_EMAIL!,
        subject: `New Contact • ${ticket}`,
        html: companyHTML
      })
    ]);

    return NextResponse.json({ ok: true, ticket });
  } catch (e:any) {
    return NextResponse.json({ ok:false, error: e.message ?? 'invalid' }, { status: 400 });
  }
}
```

`/lib/ticket.ts`:

```ts
export function genTicket() {
  const d = new Date();
  const y = d.getFullYear();
  const m = String(d.getMonth()+1).padStart(2,'0');
  const day = String(d.getDate()).padStart(2,'0');
  const num = String(Math.floor(1000 + Math.random()*9000));
  return `VTP-${y}${m}${day}-${num}`;
}
```

`/lib/mail.ts` — ابنِ HTML متوافق مع الهوية، RTL عند العربية:

```ts
type MailData = { name:string; email:string; phone?:string; message:string; ticket:string; locale?:'ar'|'en' };

const baseStyles = `
  body{margin:0;padding:0;background:#0a0a0a;color:#eee;font-family:system-ui,-apple-system,Segoe UI,Roboto}
  .wrap{max-width:640px;margin:auto;padding:32px}
  .card{background:#111;border:1px solid #222;border-radius:16px;padding:24px}
  .brand{font-weight:700;font-size:18px;letter-spacing:.4px}
  .muted{color:#bbb}
  .chip{display:inline-block;padding:6px 10px;border:1px solid #333;border-radius:999px}
`;

export function renderClientEmailHTML(d: MailData) {
  const arabic = d.locale === 'ar';
  return `
  <!doctype html><html lang="${arabic?'ar':'en'}" dir="${arabic?'rtl':'ltr'}">
  <head><meta charset="utf-8"><meta name="viewport" content="width=device-width,initial-scale=1">
  <style>${baseStyles}</style></head>
  <body><div class="wrap">
    <div class="brand">VETAP</div>
    <h1>${arabic?'تأكيد استلام طلبك':'Your request has been received'}</h1>
    <div class="card">
      <p>${arabic?'رقم التذكرة':'Ticket'}: <span class="chip">${d.ticket}</span></p>
      <p class="muted">${arabic?'سنعاود الاتصال بك قريباً.':'We will get back to you shortly.'}</p>
      <hr style="border-color:#222">
      <p>${arabic?'الاسم:':'Name:'} ${d.name}</p>
      <p>${arabic?'البريد:':'Email:'} ${d.email}</p>
      ${d.phone?`<p>${arabic?'الهاتف:':'Phone:'} ${d.phone}</p>`:''}
      <p>${arabic?'الرسالة:':'Message:'}</p>
      <pre style="white-space:pre-wrap">${d.message}</pre>
    </div>
    <p class="muted">VETAP ©</p>
  </div></body></html>`;
}

export function renderCompanyEmailHTML(d: MailData) {
  return `
  <!doctype html><html><head><meta charset="utf-8"><meta name="viewport" content="width=device-width,initial-scale=1">
  <style>${baseStyles}</style></head>
  <body><div class="wrap">
    <div class="brand">VETAP</div>
    <h1>New Contact • ${d.ticket}</h1>
    <div class="card">
      <p>Name: ${d.name}</p>
      <p>Email: ${d.email}</p>
      ${d.phone?`<p>Phone: ${d.phone}</p>`:''}
      <p>Message:</p>
      <pre style="white-space:pre-wrap">${d.message}</pre>
    </div>
  </div></body></html>`;
}
```
