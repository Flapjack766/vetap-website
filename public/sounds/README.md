# Check-in Sound Files

Place the following audio files in this directory for check-in feedback sounds:

## Required Files

1. **success.mp3** - Short, pleasant sound for valid check-ins (recommended: 0.3-0.5 seconds)
2. **warning.mp3** - Alert tone for already-used passes (recommended: 0.3-0.5 seconds)
3. **error.mp3** - Error sound for invalid/expired passes (recommended: 0.3-0.5 seconds)

## Recommended Specifications

- Format: MP3 or WAV
- Duration: 200-500ms
- Sample Rate: 44100 Hz
- Channels: Mono or Stereo
- Volume: Normalized

## Free Sound Resources

You can find free sound effects at:
- https://freesound.org
- https://soundbible.com
- https://zapsplat.com

## Notes

The check-in app will work without these files, but audio feedback improves the user experience significantly for gate staff.
